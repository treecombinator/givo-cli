# Security policy

Report vulnerabilities in the givo CLI or the GIVO registry to
`dev (at) treecombinator.com` with the subject `[givo security]`. Please do not open
public issues for exploitable problems; they get a public record once fixed.

Malicious or compromised packages on the registry: https://givo.dev/abuse. Verified bad
releases are tombstoned (downloads answer 410 and the record stays public as evidence)
and the publishing token is revoked.

Supported version: the latest release of `@givo/cli`.
